# Loan-Data-Insights-Dashboard
## Objective
The objective of this project is to leverage Power BI to visualize and analyze bank loan data by creating a comprehensive dashboard. It aims to track critical KPIs like loan applications, disbursed amounts, repayments, and borrower profiles to identify trends, improve loan strategy, and enhance decision-making.

# Summary KPIs
- Total Loan Applications  
- MTD Loan Applications  
- PMTD Loan Applications  
- Total Funded Amount  
- MTD Funded Amount  
- PMTD Funded Amount  
- Total Amount Received  
- MTD Amount Received  
- PMTD Amount Received  
- Average Interest Rate  
- MTD Average Interest Rate  
- PMTD Average Interest Rate  
- Average Debt-to-Income (DTI)  
- MTD Average DTI  
- PMTD Average DTI  
# Good Loan KPIs
- Good Loan Percentage  
- Good Loan Applications  
- Good Loan Funded Amount  
- Good Loan Amount Received
#  Bad Loan KPIs
- Bad Loan Percentage  
- Bad Loan Applications  
- Bad Loan Funded Amount  
- Bad Loan Amount Received  
# Overview KPIs
- Monthly Loan Trends  
- State-wise Loan Distribution  
- Term-wise Loan Distribution  
- Employment Length Analysis  
- Loan Purpose Breakdown  
- Home Ownership Analysis
